1 · What is a function?

Everything in this unit rests on one idea, so it comes first.

By the end: you can decide whether a set of pairs, a table or a graph is a function, and say why.

One input, one output

A function is a rule that gives exactly one output for every input. Two different inputs are allowed to share an output — what is never allowed is one input with two outputs.

Worked example

{(1,2), (2,3), (2,5)} is not a function: the input 2 is paired with both 3 and 5.
{(1,4), (2,4), (3,4)} is a function: each input appears once. The repeated output 4 is fine.

Vertical line test: on a graph, if any vertical line meets the curve more than once, one input has two outputs — not a function.
GuidedWhich relation is a function?
Q1 · Select the relation that is a function.
Look for a repeated input. In the first set the inputs are 0, 1, 2 — all different, so each has exactly one output.
PracticeMake it a function
Q2 · A relation contains (6,1), (7,4) and (6,k). For the relation to be a function, k must equal
The input 6 already has an output. A function cannot give that same input a second, different output.

Q3 · Does this graph represent a function?
The dashed vertical line crosses the curve at two points, so that one input has two different outputs. Being curved is not itself a problem — plenty of functions are curved.

2 · Showing a function four ways

Before comparing functions, you need to read them in whatever form they arrive.

By the end: you can move between a table, an equation and a description of the same function.

The same function, four ways

A function can be shown as a table, a graph, an equation, or a description in words. They are four views of one rule.

Worked example

y = 3x + 1 → when x = 0, 1, 2 the outputs are 1, 4, 7. In words: start at 1 and add 3 each step.

GuidedEvaluate an equation
Q1 · For y = 4x − 3, find y when x = 5.
PracticeContinue a table

x: 0, 1, 2  →  y: 6, 10, 14

Q2 · What is the output when x = 3?
Look at what the output does for each step of 1 in the input, then take one more step.
PracticeTable → equation
Q3 · Which equation matches the table above?
The output rises by 4 for each step of 1, so the coefficient of x is 4. At x = 0 the output is 6, so the constant is 6.

3 · Linear or non-linear?

One test settles it, and it is the same test whether you are given a table, a graph or an equation.

By the end: you can judge whether a function is linear from any representation, and justify it by rate of change.

Constant rate of change

A function is linear when the output changes by the same amount for every equal step in the input. Its graph is a straight line and it can be written y = mx + b. If the rate of change varies, the function is non-linear.

Worked example

x: 1, 2, 3 → y: 4, 7, 10. Each step of 1 in x adds 3 to y, every time — so it is linear with rate of change 3.

Watch the input steps. If the inputs do not go up in equal jumps, compare (change in y) ÷ (change in x), not the output differences alone.
GuidedSpot the non-linear one
Q1 · Which of these is not a linear function?
Squaring makes the rate of change grow: from x = 1 to 2 the output rises by 3, but from 2 to 3 it rises by 5.
PracticeWhich tables are non-linear?

Table I — in: 3, 6, 9, 12, 15 → out: 1, 4, 9, 16, 25
Table II — in: 2, 4, 6, 8, 10 → out: 19, 9, −1, −11, −21
Table III — in: 6, 3, 0, −3, −6 → out: 216, 27, 0, −27, −216

Q2 · Select every table that represents a non-linear function.
For each table divide the change in the output by the change in the input, twice. A linear function gives the same value both times.
PracticeFind the rate of change

x: 2, 5, 8  →  y: 7, 16, 25

Q3 · What is the rate of change?
The input jumps by 3 each time, not by 1. Divide the change in output by the change in input.
ApplyArea of a square

The area of a square is A = s², where s is the side length.

Part A · Find A when s = 7.
Part B · Is A a linear function of s? Type yes or no.
ExamExam-style · 8.AT.C.8 — error analysis
A student says y = 5/x is a linear function “because there is no exponent”. What is wrong with that reasoning?
Linearity is about a constant rate of change, not about whether an exponent is written down. Check: x = 1 → 5, x = 2 → 2.5, x = 4 → 1.25 — the drops are 2.5 then 1.25, so the rate changes.

4 · Reading y = mx + b from a graph

A graph is not just a picture of a function — both numbers in its equation can be read straight off it.

By the end: you can identify the slope and y-intercept from a graph, and match a graph to its equation.

Reading a line off its graph

Every straight-line graph tells you both numbers in y = mx + b directly:

  • b — the y-intercept: where the line crosses the vertical axis.
  • m — the slope: pick two points where the line crosses corners of the grid, then rise ÷ run.
Worked example

A line through (0, 2) and (3, 8): it crosses the vertical axis at 2, so b = 2. From (0,2) to (3,8) the rise is 6 and the run is 3, so m = 6 ÷ 3 = 2. The equation is y = 2x + 2.

Sign check: a line going down left-to-right has a negative slope. Read it before you compute it.
GuidedRead the y-intercept
Q1 · What is the y-intercept of this line?
PracticeRead the slope
Q2 · For the same line, what is the slope?
Use the two marked points. Count how far up the line goes, then how far across, and divide.
PracticeMatch the equation to the graph
Q3 · Which equation matches this graph?
It crosses the vertical axis at 4, so b = 4. From (0,4) to (2,0) the line falls 4 over a run of 2, so m = −2.
ApplyBoth numbers from one graph
Part A · What is the y-intercept?
Part B · What is the slope? (a fraction is fine)
ExamExam-style · 8.AT.D.9
A line passes through (0, −1) and (4, −9). Which statement about its graph is true?
The point with x = 0 is the intercept, so b = −1. The slope is (−9 − (−1)) ÷ (4 − 0) = −8 ÷ 4 = −2 — down 2 for each 1 across.

5 · Building a linear function

Given two points, a table or a story, you can write the function it describes.

By the end: you can find the rate of change and initial value from any description, and write y = mx + b.

Rate of change and initial value

To build y = mx + b from information rather than a picture:

  • Rate of change m = (change in y) ÷ (change in x).
  • Initial value b is the output when x = 0. If 0 is not in your data, work back from a point you do have.
Worked example

Given (2, 11) and (5, 20): m = (20 − 11) ÷ (5 − 2) = 9 ÷ 3 = 3. Then 11 = 3(2) + b, so b = 5 and the function is y = 3x + 5.

GuidedRate of change from two points
Q1 · A linear function passes through (0, 7) and (4, 19). What is its rate of change?
PracticeInitial value
Q2 · For the same function, what is the initial value?
The initial value is the output when the input is 0 — one of the two points already gives you that.
PracticeFrom a situation

A pool holds 500 litres and is filled at a steady 25 litres per minute.

Part A · How much water is in the pool after 12 minutes?
L
Part B · After how many minutes does it hold 1000 litres?
min
ApplyWhen the table skips zero
Q4 · A linear function passes through (3, 14) and (7, 26). What is its initial value?

ExamMCAP · 8.F.B.4

Anton has some money saved for school. To increase the amount he has saved, he got a job and added all the money earned to the amount already saved.

  • Anton was paid a constant rate in dollars per hour at his job.
  • After working 10 hours, the amount he had saved was $450.
  • After working 20 hours, the amount he had saved was $600.
  • Anton did not save any other money or spend any of the money he had saved.
Which equation represents y, the amount of money in dollars Anton would have saved after working x hours?
The rate is (600 − 450) ÷ (20 − 10) = 15 dollars per hour. Working back, 450 = 15(10) + b gives b = 300 — the money he had saved before the job.

6 · Comparing two functions

The exam almost always gives you the two functions in different forms on purpose.

By the end: you can compare the rate of change and initial value of two functions given in different forms.

Compare like with like

Two functions rarely arrive in the same form. Put both into the same terms — find each rate of change and each initial value — then compare those numbers.

Worked example

Function A from a table (0,3), (1,5) has m = 2, b = 3. Function B is y = 4x + 1, so m = 4, b = 1. B grows faster; A starts higher.

GuidedWhich grows faster?

Function P: y = 6x + 2
Function Q: the table x: 0, 2 → y: 5, 13

Q1 · Which function has the greater rate of change? Type P or Q.
PracticeWhich starts higher?
Q2 · For the same pair, which has the greater initial value? Type P or Q.
The initial value is the output at x = 0. Both are readable without any calculation.
ApplyTwo descriptions
Q3 · Function R starts at 12 and decreases by 3 at each step. Function S is y = −5x + 12. Which statement is true?
Both start at 12. R falls 3 per step and S falls 5 per step, so S falls faster. A more negative slope is a steeper fall.
ExamMCAP · 8.F.A.2

Certain values of linear function J are shown in the following table, where x is the input variable and y is the output variable.

x: 0, 1, 2  →  y: 2, 7, 12
Linear function K is represented by the equation y = 3x + 8.

Part A · The slope of the graph of function J is … the slope of the graph of function K.
J rises 5 for each step of 1, so its slope is 5. K has slope 3, and 5 > 3.
Part B · The y-intercept of the graph of function J is … the y-intercept of the graph of function K.
J has y-intercept 2 (its output at x = 0); K has 8, and 2 < 8.

7 · Describing and sketching graphs

Not every question wants a number. Some want the shape, and the reason for it.

By the end: you can describe a graph qualitatively and sketch one from a description in words.

Describing a graph in words

To describe a graph, work left to right and say, for each stretch, whether it is increasing, decreasing or constant, and whether it is linear (straight) or non-linear (curved). A sketch does not need numbers — it needs the right shape.

Worked example

A cyclist speeds up, then holds a steady speed, then stops to rest. Distance against time: a curve getting steeper, then a straight slanted stretch, then a flat stretch.

Flat means constant, not stopped-in-place on the axis. A horizontal stretch says the output is not changing.
GuidedMatch the story to the shape
Q1 · A tank is filled quickly at first, then more and more slowly, and is never emptied. Which describes the graph of volume against time?
The volume never falls, so the graph never goes down. “More and more slowly” means the steepness keeps changing, so it is a curve, not a straight line.
PracticePlot two points on the model

A candle is 12 cm tall and burns down 2 cm every hour.

Step 1 · Plot the point showing the candle’s height after 4 hours.
Step 2 · Plot the point at which the candle has burnt out.
PracticeIncreasing, decreasing, constant
Q3 · A graph rises steeply, then rises gently, then is horizontal. Which is true of its rate of change?
Rising means a positive rate of change; steeper means larger. A horizontal stretch means the output is not changing at all, so the rate is zero — not negative.
ReasoningDescribe the graph you would draw
Q4 · A company reviews its sales for one year. Sales rose steadily for the first two months, then declined steadily over the next four, stayed constant for three, and rose steadily for the last three. Describe the graph you would sketch of sales against time: name each stretch as increasing, decreasing or constant, and say whether each is linear or non-linear. Then suggest one thing that might explain the shape.
ExamMCAP · 8.F.B.5

The graph of a function is shown in the following coordinate plane.

Over which interval does the graph show a constant rate of change?
A constant rate of change is a straight stretch. Only P to Q is a straight segment; the other three are curved, so their steepness keeps changing.

Rate of change is (change in y) ÷ (change in x). Ada compared the outputs but forgot that the inputs jump by 3, not 1. The function is linear — both steps give −2.
